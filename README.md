# Angel Assist AI

AI-powered internal support assistant for customer support and operations teams.

## Objective
Reduce support handling time and improve operational efficiency through AI-assisted SOP guidance and troubleshooting support.

## Features
- SOP assistance
- TAT guidance
- Escalation workflows
- Support query handling

## Categories
- Funds
- KYC
- IPO
- Trading
- UPI
- Account Opening

## Demo Queries
- Fund not showing after payin
- Mobile update TAT
- UPI mandate failed

- Bank processing delay
- UPI mandate pending
- Settlement delay

Suggested Steps:
- Verify bank debit status
- Retry after some time
- Check bank server status

Escalation:
Payments Operations

24–48 working hours
